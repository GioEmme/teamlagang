HERO IMAGES — public/hero/
============================

Droppa qui le immagini che vuoi nella sequenza hero della home.

Regole:
- Formati: .jpg, .jpeg, .png, .webp, .avif
- Ordine: per nome file (alfabetico). Usa prefisso numerico per decidere:
    01-ingresso-pista.jpg
    02-macchina-touring.jpg
    03-partenza-gara.jpg
    ...
- Dimensioni consigliate: 2400x1600 px (o ratio 3:2), < 500 KB (comprime con tinypng / squoosh)
- Minimo 1 immagine. Ottimo 4–8.

Il sito legge la cartella automaticamente: basta droppare e ricaricare.

Se la cartella è vuota, la home mostra un hero di fallback.
